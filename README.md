# Balancer ETH/WETH Wrapper

This is an implementation of a ETH/WETH wrapper contract for Argent.

## Usage

### Install dependencies:

```
npm install
```

### Create `.env` file:

```
MNEMONIC=<Your 12 word mnemonic phrase>
```

### Compile and run tests

```
truffle compile
npm run kovan-test
```
